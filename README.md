# AplicacionPython

- Programa que realiza un registro de productos según su categoría y precio en una lista de diccionarios.
- Se utilizaron las librerías de requests y json para el guardado, carga de archivos y para realizar la solicitud a la API para obtener el valor del dolar actual.
- El menú incluye el ingreso del artículo, visualización de cada producto registrado, eliminar artículo por índice, carga de archivos, finalizar el programa y guardar los datos de la sesión. 👍
